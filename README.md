# dotspace
Dotty for Space code
Written by Lucy Whalley (github: lucydot) to trial and analyse Dot-Space 
drawings by Richard Scott (richard-md-scott.tumblr.com).   

Dependancies are: numpy, matplotlib, iPython, 
argparse, time.
